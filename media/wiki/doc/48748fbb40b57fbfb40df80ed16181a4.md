# 环境搭建
> 由于业务原因，需要提高开发和测试之间的效率，我们需要一套稳定的开发环境以及一套稳定的测试环境

## 搭建开发环境
### 前端
```
yum -y install nginx
```
### 后端
```
yum -y install java-8
yum -y install tomcat
```
### 消息队列
```
yum -y install mq
```
### 数据库
```
docker-compose up -d 
```
